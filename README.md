# Kafka Spring Boot Consumer & Producer

Kafka consumer and producer implementation with Spring Boot.

## Tech Stack

- **Framework:** Spring Boot
- **Messaging:** Apache Kafka
- **Language:** Java
- **Build:** Maven

## Features

- Kafka producer for publishing messages
- Kafka consumer for consuming messages
- Topic configuration and management

## Getting Started

```bash
# Start Kafka (Docker)
docker-compose up -d

# Run the application
mvn spring-boot:run
```
